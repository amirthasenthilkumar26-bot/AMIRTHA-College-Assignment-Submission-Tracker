 College Assignment Submission Tracker

A simple and clean React dashboard designed to help track student assignment submissions in real-time.

Features
* Add new assignments with titles, subjects, and deadlines.
* Live statistics tracker (Total, Completed, and Pending tasks).
* Quick action buttons to change status or delete assignments.

Built With
* React.js (Vite)
* JavaScript / JSX
* Inline CSS

How to Run Locally
1. Run `npm install` to download dependencies.
2. Run `npm run dev` to start the local server.
3. Open `http://localhost:5173` in your browser.
